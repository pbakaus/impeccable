# Impeccable (Cursor plugin)

Design fluency for frontend development, packaged as a native Cursor plugin.

This plugin bundles:

- **Skill** `impeccable` — `/impeccable` with 23 commands (polish, audit, critique, craft, and more) plus curated anti-pattern guidance.
- **Agent** `impeccable-manual-edit-applier` — applies leased live copy-edit batches to source.
- **Hook** — a `preToolUse` design gate that runs the Impeccable detector on proposed UI writes and **blocks** anti-pattern writes before they land.

## Install

Install from the Cursor marketplace, or load it locally:

```bash
cp -r plugin-cursor ~/.cursor/plugins/local/impeccable
# Restart Cursor, or run "Developer: Reload Window"
```

Verify the skill, agent, and hook under **Customize**.

## Notes

- This directory is generated by the Impeccable build (`bun run build:release`). Do not edit it by hand; change `skill/` and the build sources instead.
- Homepage: https://impeccable.style
