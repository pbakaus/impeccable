# Origin brief

This is the prompt that produced `index.html` — a one-shot from Claude with default
design instincts and no skill loaded.

> Build me a landing page for a SaaS product called Lumina. It's an AI-native
> workflow tool for product teams. Make it feel modern, warm, editorial, with
> great typography. Include a hero, three feature highlights, and a closing CTA.
> Single HTML file, self-contained.

## What this page demonstrates

Without any guidance, current frontier models converge on a recognizable aesthetic:

- Fraunces (or similar humanist serif) for headlines
- Warm cream background, subtle peach gradient
- Burnt-orange or terracotta accent
- Three icon-above-h3 feature tiles with generic copy ("Lightning Fast", etc.)
- Rounded-everything, soft shadows, polite spacing

Twelve months ago this looked novel. Today every freshly-shipped SaaS landing
page looks like this. The new defaults are the new slop.

## Demo use

Open in a browser, then run `/impeccable critique` to surface the patterns
the detector flags. Then run `/impeccable live` to iterate on the hero or
feature tiles in place.
